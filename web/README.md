# Ferelix Web

A web client for self-hosted media servers (similar to Plex/Jellyfin).

## Overview

Ferelix Web serves two purposes:
- **Web Client**: A Vue 3 SPA for browsing and playing media in any modern browser

## Quick Start

### Development

```bash
# Install dependencies
pnpm install

# Run web client dev server
pnpm dev

# Build web client
pnpm build
```
